# TeacherRating

## Жоба атауы
TeacherRating — мұғалімдерді бағалау платформасы

## Мақсаты
Қазақстандық студенттерге мұғалімдерді объективті бағалауға 
мүмкіндік беру. Платформада іздеу, сүзгілеу, бағалау, 
таңдаулылар және аналитика функциялары бар.

## Қолданылатын модель
**Agile (Скрам)** — икемділік, тез нәтиже, кері байланыс

## Технологиялар
- Frontend: HTML5, CSS3, JavaScript
- Backend: Supabase (PostgreSQL)
- Хостинг: Vercel
